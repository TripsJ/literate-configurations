* About this Configuration

This literate shell configuration maintains both Bash and Zsh from a single Org file.
Code is organized by topic, shared between shells using noweb references, with shell-specific blocks where needed.

Structure:
- *Shared* blocks: Used by both Bash and Zsh (aliases, functions, environment variables)
- *Shell-Specific* blocks: Bash-only or Zsh-only configurations (options, completion settings)

To regenerate the shell configs, run:
  =C-c C-v t= (in Emacs)
  or =org-babel-tangle-file "shellrc.org"=

** Repository Structure

This org file is part of a literate config repository:

#+begin_example
dotfiles/
├── shellrc.org           (this file - generates both .bashrc and .zshrc)
├── .bashrc              (tangled output)
├── .zshrc               (tangled output)
├── emacs-init.org       (your Emacs config)
└── README.md
#+end_example

** Key Org-Babel Concepts Used

- =:tangle FILE= - specifies which file(s) to write the code block to
- =:noweb yes= - enables noweb reference expansion (=<<ref-name>>=)
- =#+name:= - labels a code block for noweb reference
- =#+noweb-ref:= - attaches content to a named reference for inlining

** File Generation Notes

After editing this file, tangle with =C-c C-v t= to generate:

- ~/.zshrc

* Functions

Reusable shell functions for common tasks.

** Shared Functions

#+name: shared-functions
#+begin_src bash
# Make and cd into a directory
mcd() {
  mkdir -p "$1" && cd "$1"
}

# Find files matching a pattern
fif() {
  if [ -z "$1" ]; then
    echo "Usage: fif <pattern>"
    return 1
  fi
  find . -type f -name "*$1*"
}

# Search in files
# Usage: search "pattern" [path]
search() {
  local pattern="$1"
  local path="${2:-.}"
  grep -r --color=auto "$pattern" "$path"
}

# Create a backup of a file
backup() {
  if [ -z "$1" ]; then
    echo "Usage: backup <file>"
    return 1
  fi
  cp "$1" "$1.backup"
  echo "Backed up $1 to $1.backup"
}

# Show disk usage of current directory
duh() {
  du -sh * | sort -rh
}
#+end_src

** Bash-specific Functions

#+begin_src bash :tangle ~/.bashrc
<<shared-functions>>

# Extract archive files
extract() {
  if [ -f "$1" ]; then
    case "$1" in
      *.tar.bz2) tar xjf "$1" ;;
      *.tar.gz)  tar xzf "$1" ;;
      *.tar.xz)  tar xJf "$1" ;;
      *.bz2)     bunzip2 "$1" ;;
      *.rar)     unrar x "$1" ;;
      *.gz)      gunzip "$1" ;;
      *.tar)     tar xf "$1" ;;
      *.tbz2)    tar xjf "$1" ;;
      *.tgz)     tar xzf "$1" ;;
      *.zip)     unzip "$1" ;;
      *.Z)       uncompress "$1" ;;
      *.7z)      7z x "$1" ;;
      *)         echo "Cannot extract '$1'" ;;
    esac
  else
    echo "'$1' is not a valid file"
  fi
}

* Reference & Maintenance

** Quick Reference

| Command | Effect |
|---------+--------|
| ~C-c C-v t~ | Tangle current org file (regenerate .bashrc & .zshrc) |
| ~M-x org-babel-tangle-file~ | Tangle with file selection |
| ~source ~/.bashrc~ | Reload bash configuration |
| ~source ~/.zshrc~ | Reload zsh configuration |

** Troubleshooting

- **Changes not taking effect**: Run =C-c C-v t= to retangle, then reload your shell (~source ~/.bashrc~ or ~source ~/.zshrc~)
- **Syntax errors after tangle**: Check the tangled output files (.bashrc/.zshrc) for the line numbers reported
- **Org file won't tangle**: Ensure =org-babel-tangle= is available (it's built-in to modern Org)
- **Losing manual edits**: Never edit .bashrc/.zshrc directly; always edit the .org file and retangle
